Key Changes

What Changes

  • Sidebar: Vibrancy material, section headers, badge counts, refined folder hierarchy
  • Document rows: File-type colored icons with gradients, two-line layout (name + metadata), subtle hover/selection states
  • Detail panel: Large file icon with shadow, pill-shaped action buttons, refined tag chips, card-style preview with shadow
  • Typography: Tighter letter-spacing, consistent font weights, semantic hierarchy
  • Spacing: Consistent 8pt grid, generous padding, visual breathing room

What Stays the Same

  • Three-column NavigationSplitView layout
  • All existing functionality (import, checkout, versions, etc.)
  • Toolbar actions and menu commands
  • Settings, Help, About dialogs
  • Folder tree, search, filtering
