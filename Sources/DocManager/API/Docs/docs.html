PandyDoc API Reference

Base URL: http://127.0.0.1:8080/api/v1

GET/health
Check server health and status.
Response Example
{"status":"ok","version":"1.0.0","uptime":123.45}
GET/documents
List documents with pagination and filtering.
Parameters
page: integer (default: 1)
limit: integer (default: 20, max: 100)
sort: name | created_at | updated_at
order: asc | desc
folder_id: uuid (optional)
Response Example
{"data":[],"total":0,"page":1,"limit":20,"has_more":false}
POST/documents
Create a new document.
Request Body
{"name":"My Document","content":"...","folder_id":"uuid"}
GET/documents/:id
Get a document by ID.
PUT/documents/:id
Update a document.
DELETE/documents/:id
Delete a document.
POST/documents/:id/check-in
Check in a document.
POST/documents/:id/check-out
Check out a document.
GET/documents/:id/versions
List all versions of a document.
GET/documents/:id/versions/:version
Get a specific version.
POST/documents/:id/restore/:version
Restore document to a specific version.
GET/folders
List all folders.
POST/folders
Create a new folder.
GET/folders/:id
Get a folder by ID.
PUT/folders/:id
Update a folder.
DELETE/folders/:id
Delete a folder.
GET/templates
List all templates.
POST/templates
Create a new template.
GET/templates/:id
Get a template by ID.
PUT/templates/:id
Update a template.
DELETE/templates/:id
Delete a template.
GET/search
Search documents by query string.
GET/settings
Get API settings.
PUT/settings
Update API settings.
